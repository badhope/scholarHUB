# 页面设计参考

## 1. 全局氛围

整体走向"印刷学术期刊" + "Apple 产品页" 的混合调性。摒弃所有花哨渐变、毛玻璃、彩虹色、AI 化的卡片阴影。视觉重点永远在 **文字** 与 **留白**。

### 1.1 颜色变量

```css
--paper:    #f8f6f1;   /* 纸白，浅色模式背景 */
--ink:      #1a1a1a;   /* 墨黑，正文与标题 */
--ink-soft: #3c3c3c;   /* 次级文字 */
--rule:     #d8d4c7;   /* 描边、分割线 */
--moss:     #2f4f3a;   /* 墨绿，强调按钮与链接 */
--ochre:    #a86b3c;   /* 赭石，标签与标记 */
--night:    #15171a;   /* 深色模式背景 */
--night-ink:#e8e4d8;   /* 深色模式正文 */
```

### 1.2 字体

| 用途 | 字体 | 字号 |
|---|---|---|
| H1 (Hero) | Cormorant Garamond 600 | clamp(2.5rem, 6vw, 4.5rem) |
| H2 | Cormorant Garamond 500 | 2rem |
| H3 | EB Garamond 500 | 1.5rem |
| 正文 | EB Garamond 400 | 1.125rem (18px) |
| 中文 | Noto Serif SC 400 | 同上 |
| 标签 / 元数据 | JetBrains Mono 400 | 0.8125rem (13px) |
| 按钮 | EB Garamond 500 | 0.9375rem |

行高：标题 1.2，正文 1.8，列表 1.6。

### 1.3 留白规则

- 主内容区最大宽 `880px`，左对齐到网格的 1/3 处（不是居中），营造"出版物的左栏"质感。
- 区块上下间距至少 `120px`。
- 卡片内边距 `24px`，卡片间距 `40px`。
- 任何元素周围保持至少 `40px` 真空带。

## 2. 页面

### 2.1 首页 `/`

**结构**：
1. 顶部一条 1px 横线（`--rule`），左侧小字 "ScholarHUB — Vol. 1 · 2026"。
2. Hero：标题 *An open shelf of papers, books, and datasets.*，副标题一行中文，搜索框（极细描边，输入时描边由纸白变墨绿）。
3. 学科卡片：6 个并排卡片，每个卡片仅一行衬线大字与一行小字说明（学科名 / Subdisciplines: N）。点击后向下展开，0.3s 缓动，下方出现该学科下前 6 个资源卡片 + "View all" 链接。
4. 精选资源：横向滚动列，每个卡片缩略 1 行摘要 + 引用次数小标。
5. 页脚：1px 横线 + 三列：导航、贡献、版权（全部用小字 JetBrains Mono）。

### 2.2 资源列表 `/resources`

**结构**：
1. 顶部一行可滚动 chips：全部、论文、教材、数据集、教程。
2. 下方按年份倒序排列的卡片列表。
3. 每张卡片包含：标题（衬线，hover 时变墨绿下划线）、作者+年份（13px 等宽）、摘要预览（2 行截断）、3-5 个标签（赭石描边胶囊）、右下角"下载 / 跳转"。

### 2.3 资源详情 `/resource/:id`

**结构**：
1. 顶部面包屑：学科 > 资源。
2. 大标题（衬线 H1）。
3. 元数据条：作者 · 年份 · 期刊 · DOI（DOI 用墨绿下划线链接）。
4. 摘要全文（首行缩进 2em，行高 1.8）。
5. 操作条：四个等宽按钮 - 跳转下载、跳转 DOI、复制引用、加入收藏。
6. 引用生成下拉：4 种格式（APA / MLA / GB/T 7714 / BibTeX），点击复制后出现"已复制"小提示 1.5s 后消失。
7. 相关推荐：同学科或同作者的其他资源，3 张。

### 2.4 学科分类页 `/discipline/:slug`

与列表页同结构，但顶部加入学科简介、覆盖子领域、相关数据集数量。

### 2.5 搜索页 `/search?q=...`

- 顶部显示查询关键词与结果数。
- 下方按相关度排序的卡片列表。
- 空结果时显示"未找到相关资源，试试..." 提示。

### 2.6 收藏夹 `/favorites`

- 顶部一行提示"收藏仅保存在本地浏览器，清除浏览器数据将丢失"。
- 卡片列表，每张卡片左侧多一个删除按钮。
- 提供"导出 JSON" 与"清空全部" 按钮。

### 2.7 设置 `/settings`

- 主题：浅色 / 深色 / 跟随系统（单选条）。
- 动效强度：完整 / 减弱 / 关闭。
- 字体大小：标准 / 加大。
- 底部"重置所有设置" 链接。

### 2.8 关于 `/about`

- 项目理念、贡献指南、引用本项目的方式、致谢。

## 3. 微动效

| 场景 | 动效 | 时长 | 缓动 |
|---|---|---|---|
| 卡片展开 | max-height 0 → auto + opacity 0 → 1 | 0.3s | ease-out |
| 卡片 hover | border-color 浅灰 → 墨黑 | 0.2s | linear |
| 页面切换 | opacity 1 → 0 → 1（淡入淡出） | 0.25s | ease-in-out |
| 收藏动作 | 按钮内出现 1 个 ✓，1.2s 后消失 | 0.2s | ease-out |
| 复制提示 | 底部小气泡 fade-in 1.2s → fade-out | 0.2s + 1.2s 保持 | ease-out |
| 主题切换 | 整页颜色 transition | 0.4s | ease-in-out |

设置中"减弱" 时长减半，"关闭"则全部 0s。

## 4. 资源卡片设计细节

```
┌──────────────────────────────────────────┐
│  Title in Cormorant Garamond, 1.5rem     │
│                                          │
│  Authors · Year · Venue  (mono, 13px)    │
│                                          │
│  Preview text in EB Garamond, 17px,      │
│  line-height 1.6, max 2 lines ellipsis. │
│                                          │
│  [#深度学习] [#神经网络] [#textbook]    │
│                                          │
│                            [↓ Download]  │
└──────────────────────────────────────────┘
   1px border --rule, hover border --ink
   24px padding
```

## 5. 关键设计反 AI 化策略

- **不使用** 任何渐变色、彩色阴影、毛玻璃、霓虹、emoji 装饰。
- **不使用** 居中大圆角按钮。
- **不使用** 通用蓝色 (`#3b82f6`) 或紫色 (`#8b5cf6`)。
- **使用** 1px 极细描边、低饱和度、衬线字体、非居中布局、左对齐的"出版感"。
- 重要数字、版本号、年份用 JetBrains Mono，呈现"印刷品的索引号"质感。
